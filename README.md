## 🛠️ 环境准备

为了确保代码顺利运行，建议准备以下环境：

1.  **硬件**: ESP32 开发板（如 ESP32-S3, ESP32-C3 或经典款 ESP32）。
2.  **IDE**: 
    - [Arduino IDE 2.x](https://www.arduino.cc/en/software) (安装 ESP32 开发包)。
    - 或者 VS Code + [PlatformIO](https://platformio.org/)。
3.  **库支持**:
    - `I2Cdev` / `MPU6050` (用于 L25/L26)。
    - `NimBLE-Arduino` (用于 L26 BLE)。
