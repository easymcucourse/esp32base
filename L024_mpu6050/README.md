# 🧭 ESP32 教程：MPU6050 全数据读取与 DMP 姿态解算

欢迎来到 **轻松易学嵌入式** 频道！本期视频我们将深入探索 MPU6050 六轴传感器，利用其内置的 DMP (Digital Motion Processor) 轻松获取高精度的姿态数据。

🎥 **本集视频教程**: [点击观看](https://www.youtube.com/watch?v=vLiZccZTGUQ)

---

## 🛠️ 项目主要功能

1.  **DMP 姿态解算**: 直接读取 MPU6050 内部解算的四元数，并转换为直观的 **欧拉角 (Yaw, Pitch, Roll)**，无需在主控端进行复杂的卡尔曼滤波。
2.  **全数据监控**: 能够同时读取传感器所有的核心数据：
    *   **姿态角**: 偏航 (Yaw)、俯仰 (Pitch)、横滚 (Roll)
    *   **世界坐标加速度**: 去除重力影响并转换到世界坐标系的运动加速度
    *   **原始数据**: 加速度计 (Accel) 和 陀螺仪 (Gyro) 的原始 ADC 值
3.  **串口实时输出**: 格式化的串口打印，方便使用串口绘图仪 (Serial Plotter) 观察波形或进行数据分析。

## 🚀 快速开始

### 1. 硬件连接
请将 MPU6050 连接至 ESP32 的 I2C 引脚：

*   **ESP32 SDA (GPIO 8)**  <-->  **MPU6050 SDA**
*   **ESP32 SCL (GPIO 9)**  <-->  **MPU6050 SCL**
*   **VCC**                 <-->  **3.3V**
*   **GND**                 <-->  **GND**

*(注：引脚定义可在代码 `Wire.begin(8, 9)` 中修改)*

### 2. 软件库准备
本项目需要安装以下 Arduino 库：
*   **I2Cdev**
*   **MPU6050** (Electronic Cats 版本或 jrowberg 版本均可)

### 3. 使用说明
1.  下载并上传代码 `L024_mpu6050.ino` 到开发板。
2.  保持 MPU6050 **水平静止** 放置。
3.  打开串口监视器 (波特率 **115200**)。
4.  复位开发板，等待初始化完成 (约几秒钟)。
5.  观察串口输出的数据流。

## 📊 数据输出格式说明

程序通过串口输出制表符分隔的数据，格式如下：

```text
ypr [yaw, pitch, roll]  world [x, y, z]  acc [ax, ay, az]  gyro [gx, gy, gz]
```

*   **ypr**: 欧拉角 (单位：度)
    *   **Yaw**: 偏航角 (Z轴旋转)
    *   **Pitch**: 俯仰角 (Y轴旋转)
    *   **Roll**: 横滚角 (X轴旋转)
*   **world**: 世界坐标系下的线性加速度 (去除重力)
*   **acc**: 加速度计原始值 (LSB)
*   **gyro**: 陀螺仪原始值 (LSB)
