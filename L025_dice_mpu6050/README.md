# 🎲 ESP32 实战：Web 版六面重力骰子 (MPU6050 + WebSocket)

欢迎来到 **轻松易学嵌入式** 频道！这是一个综合性实战项目，将传感器数据采集、高精度解算和网页 3D 渲染完美结合。

---

## 🛠️ 项目核心特性

1.  **高精度姿态解算**: 利用 MPU6050 的硬件 DMP (Digital Motion Processor) 输出四元数，彻底告别欧拉角死锁（Gimbal Lock）。
2.  **Web 3.0 控制台**: 内置 Web 服务器，通过 LittleFS 挂载网页文件。使用 Three.js 在浏览器中实时同步 3D 骰子状态。
3.  **零贴装误差模式**: 支持“一键同步”功能，无论传感器如何安装，都能在软件层面瞬间校准坐标系。
4.  **持久化存储**: 硬件偏移量（Calibration Offsets）自动保存至 Flash (NVS)，断电不丢失。

---

## 🚀 快速开始

### 1. 硬件连接
- **传感器**: MPU6050
- **引脚**: SDA=8, SCL=9 (可在代码中根据板型修改)

### 2. 软件准备
- 安装 `I2Cdev` 和 `MPU6050` 库。
- 确保已安装 ESP32 文件系统上传工具 (LittleFS)。

### 3. 下载与上传
- 使用 [arduino-esp32littlefs-plugin](https://github.com/lorol/arduino-esp32littlefs-plugin) 将 `data` 文件夹内的 `index.html` 上传至开发板。
- 编译并上传 `L25_dice_mpu6050.ino`。

---

## 🎮 使用说明

1.  **连接 WiFi**: 搜索并连接 SSID：`easymcu`（密码见代码）。
2.  **打开网页**: 在浏览器访问串口监视器输出的 IP 地址。
3.  **同步坐标**:
    - 将实物骰子 1 面向上，2 面朝向自己。
    - 点击网页上的“一键同步”按钮。
4.  **开始玩耍**: 任意旋转实物骰子，网页中的 3D 骰子将实时跟随。
